## Cluster: excalidraw__actions__3

This cluster covers seven action-definition files from `packages/excalidraw/actions/`. Each file registers one or more `Action` objects via the `register()` factory. An Action's core shape is `{ name, label, icon?, trackEvent, perform(elements, appState, value, app) => ActionResult | false, keyTest?, predicate?, checked?, viewMode?, PanelComponent? }`. The `perform` return shape is `{ elements?, appState?, captureUpdate, ... }` where `captureUpdate` is one of `CaptureUpdateAction.IMMEDIATELY` (record undo step now), `EVENTUALLY` (defer/coalesce into history), or `NEVER` (no undo entry). `PanelComponent` is a React component that renders the property panel UI and calls `updateData(value)` to re-dispatch the action with a payload.
